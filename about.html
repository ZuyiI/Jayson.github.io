About Me

I’m Jayson Lim a data enthusiast and developer with experience in analytics, data modeling, AI-integrated simulations, and back-end development. I’m passionate in any anything data related.

As seen in my project webpage, I have experience in doing data-related projects such as exploratory data analysis, data cleaning, ETL pipelines, visualization using tableau and many more. Whether it’s uncovering hidden patterns, engineering intelligent systems, or presenting insights in a meaningful way, I thrive on turning raw data into real-world value. I’m a data enthusiast driven by curiosity, always looking beyond surface-level trends to extract deeper, actionable insights. My goal is to make a measurable impact in any field I work in.

I have worked on projects that combined backend engineering, machine learning, and user-focused design. My recent work includes developing a Virtual Patient Simulation system in collaboration with the Ateneo School of Medicine and Public Health (ASMPH), where I integrated OpenAI’s API, refined mixed-reality learning tools, handled simulated patient data, and applied prompt engineering to improve AI behavior to simulate realistic patient conversations.

I'm knowledgeable with a wide range of technologies including Python, Java, C++, Django, Flask, JavaScript, HTML/CSS, and RESTful APIs for web development; PostgreSQL, MySQL, MongoDB, and DynamoDB for data management; and tools like Tableau, Tableau Prep, Pandas, NumPy, and Matplotlib for analytics and visualization. I also have experience with machine learning tools such as Scikit-learn, Google AutoML, OpenAI API, LLM fine-tuning, and prompt engineering, alongside building ETL pipelines, web scraping with Selenium, cloud services like AWS and Google Cloud, and version control using Git/GitHub.

I'm always looking to grow as a data enthusiast and make a positive impact through tech.
